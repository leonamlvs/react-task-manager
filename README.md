# React Task Manager

A simple task management application built with **React**, **Tailwind CSS**, and **React Router**. I created this project to practice the basics of building a React app, such as handling state, using basic routing, and saving data to `localStorage`.

## 🚀 Features

- **Task CRUD Operations:** Add, view, and delete tasks.
- **Task Completion:** Toggle task status with visual feedback (line-through styling).
- **Persistent Storage:** Integrated with `localStorage` to ensure your tasks are saved even after refreshing the page.
- **Dynamic Routing:** A dedicated details page for each task using `react-router-dom` and URL search parameters.
- **Form Validation:** Basic validation to ensure tasks are created with both a title and a description.
- **Responsive Design:** Fully responsive UI built with Tailwind CSS, optimized for different screen sizes.
- **Modern UI Icons:** Accessible and clean iconography using Lucide React.

## 🛠️ Technologies Used

- **React:** Functional components and Hooks (`useState`, `useEffect`, `useNavigate`, `useSearchParams`).
- **React Router:** For seamless navigation between the main dashboard and task details.
- **Tailwind CSS:** For modern, utility-first styling.
- **Lucide React:** For a consistent and professional icon set.
- **Vite:** As the build tool for a fast and optimized development environment.

## 📦 How to Run

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/leonamlvs/react-task-manager.git
   ```

2. Navigate to the project directory:

   ```bash
   cd react-task-manager
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

### Running the Project

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
